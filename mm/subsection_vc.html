မြင့်မားသော ဗီတာမင် C ဆေးရည်ငွေ့
မိမိကိုယ်တိုင်ကုန်ကျစရိတ်ကျခံရသောကုသမှု | ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးခြင်း၊ ကိုယ်ခံအားမြှင့်တင်ခြင်း၊ အသားအရေတောက်ပစေခြင်း၊ collagen ထုတ်လုပ်မှု
သာမန်အစားအစာ (သို့) ဖြည့်စွက်စာများဖြင့် သွေးတွင်းအာနိသင်ကို သိသိသာသာ မြှင့်တင်၍ မရသော ဗီတာမင် C ကို ဆေးရည်ငွေ့ဖြင့် မြင့်မားသောအာနိသင်ဖြင့် တိုက်ရိုက် ထောက်ပံ့ပေးသည်။ သွေးတွင်း ဗီတာမင် C အာနိသင်သည် အတိုင်းအတာတစ်ခု (0.4mM ခန့်) ကျော်လွန်သွားပါက ကင်ဆာဆဲလ်များတွင် hydrogen peroxide ထုတ်လုပ်၍ ရွေးချယ်ကာ ကင်ဆာဆန့်ကျင်သောအကျိုးသက်ရောက်မှုကို ပေးစွမ်းကြောင်း သတင်းပို့ထားပြီး၊ ကမ္ဘာတစ်ဝှမ်းရှိ ဆေးခန်းများတွင် ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးရန် အသုံးပြုနေကြသည်။

အလှအပ၊ ကိုယ်ခံအားနှင့် ပင်ပန်းနွမ်းနယ်မှုပြန်လည်ကောင်းမွန်ခြင်းအတွက် အနည်းငယ်မှအလယ်အလတ်ပမာဏမှသည် ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးရန်အတွက် မြင့်မားသောပမာဏ (25g-75g) အထိ သင့်ရည်မှန်းချက်နှင့်ကိုက်ညီသော ပမာဏကို အကြံပြုပါသည်။
⚠️ ဤစာမျက်နှာတွင်ဖော်ပြထားသော ကုသမှုသည် မိမိကိုယ်တိုင်ကုန်ကျစရိတ်ကျခံရသော ကုသမှု (အာမခံမလွှမ်းခြုံပါ) ဖြစ်သည်။ G6PD ချို့တဲ့သောလူနာများအတွက် ပေးအပ်၍မရပါ (hemolytic anemia အန္တရာယ်ရှိသည်)။ မြင့်မားသောပမာဏမပေးအပ်မီ G6PD စစ်ဆေးမှုကို အမြဲတမ်း ပြုလုပ်ပါသည်။ ကျောက်ကပ်ပျက်စီးခြင်း (သို့) သွေးဆေးနေသောလူနာများလည်း ကျွန်ုပ်တို့နှင့် တိုင်ပင်သင့်သည်။ ရလဒ်များသည် လူတစ်ဦးနှင့်တစ်ဦး ကွဲပြားနိုင်ပါသည်။
🍋
မြင့်မားသော ဗီတာမင် C ၏ အဓိကအကျိုးသက်ရောက်မှု
ကင်ဆာကာကွယ်ခြင်း · ကိုယ်ခံအား · အလှအပ · Collagen

ဗီတာမင် C သည် ရေပျော် antioxidant ဗီတာမင်တစ်ခုအဖြစ် ကျယ်ကျယ်ပြန့်ပြန့်လူသိများသော်လည်း၊ ဆေးရည်ငွေ့ဖြင့် မြင့်မားသောအာနိသင် ရောက်ရှိသွားပါက သာမန်ခံတွင်းမှသောက်ခြင်းဖြင့် ရရှိနိုင်မည်မဟုတ်သော သီးခြားအကျိုးသက်ရောက်မှုများ ပေါ်ထွက်လာသည်။

✓
ရွေးချယ်ကာ ကင်ဆာဆန့်ကျင်သောအကျိုးသက်ရောက်မှု (မြင့်မားသောပမာဏတွင်): သွေးတွင်း hydrogen peroxide ထုတ်လုပ်၍ catalase လုပ်ဆောင်ချက်နည်းသော ကင်ဆာဆဲလ်များကို ရွေးချယ်ကာ ထိခိုက်စေသည်။ ပုံမှန်ဆဲလ်များအပေါ် အဆိပ်သင့်မှုနည်းပါးပြီး ခီမိုကုသမှုနှင့် ကောင်းစွာပေါင်းစပ်နိုင်သည်။
✓
ကိုယ်ခံအားမြှင့်တင်ခြင်း: NK ဆဲလ်များ၊ neutrophil များနှင့် lymphocyte များ၏ လုပ်ဆောင်ချက်ကို မြှင့်တင်ပေးပြီး ဗိုင်းရပ်စ်နှင့် ဘက်တီးရီးယားများကို ခုခံနိုင်စွမ်းကို အားကောင်းစေသည်။ ထပ်ခါထပ်ခါ ပိုးဝင်ခြင်းနှင့် လွယ်ကူစွာ ပင်ပန်းနွမ်းနယ်ခြင်းအတွက် ထိရောက်သည်။
✓
Collagen ထုတ်လုပ်မှုမြှင့်တင်ခြင်း: Collagen ဖြစ်ပေါ်စေရန် မရှိမဖြစ်လိုအပ်သော coenzyme အဖြစ် လုပ်ဆောင်ပြီး အသားအရေခိုင်မာမှု၊ ပျော့ပျောင်းမှုနှင့် ဒဏ်ရာပျောက်ကင်းမှုကို မြှင့်တင်ပေးသည်။ အသားအရေအလှနှင့် အသက်ကြီးမှုနှေးကွေးစေခြင်းအတွက် အခြေခံအုတ်မြစ်ဖြစ်သည်။
✓
Antihistamine နှင့် ရောင်ရမ်းမှုဆန့်ကျင်သောအကျိုးသက်ရောက်မှု: ဓာတ်မတည့်ခြင်းလက္ခဏာများနှင့် ရောင်ရမ်းမှုအခြေအနေများကို သက်သာစေသည်။ ဖျားနာခြင်းနှင့် atopic dermatitis အတွက်လည်း ထိရောက်သည်ဟု ယူဆကြသည်။
✓
ပင်ပန်းနွမ်းနယ်မှုပြန်လည်ကောင်းမွန်ခြင်းနှင့် စိတ်ဖိစီးမှုခံနိုင်ရည်: adrenal cortex ဟော်မုန်း (cortisol) ဖြစ်ပေါ်စေရန် ကူညီပေးပြီး စိတ်ဖိစီးမှုတုံ့ပြန်ခြင်းနှင့် ပင်ပန်းနွမ်းနယ်မှုပြန်လည်ကောင်းမွန်ခြင်းကို ပံ့ပိုးပေးသည်။
ဤကုသမှုကို အကြံပြုအပ်သူများ ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးခြင်း ခီမိုကုသမှုစဉ် ဘေးထွက်ဆိုးကျိုးလျှော့ချခြင်း ကိုယ်ခံအားမြှင့်တင်ခြင်း အသားအရေတောက်ပစေခြင်းနှင့် collagen ဖြည့်တင်းခြင်း နာတာရှည်ပင်ပန်းနွမ်းနယ်မှုနှင့် နေမကောင်းခြင်း ဓာတ်မတည့်ခြင်းနှင့် ရောင်ရမ်းလွယ်သောကိုယ်ခန္ဓာသဘော အသက်ကြီးမှုနှေးကွေးစေခြင်း

High-dose vitamin C: A promising anti-tumor agent, insight from mechanisms, clinical research, and challenges

Capital Medical University (Beijing) ၏ ခွဲစိတ်ဆရာဝန်အဖွဲ့မှ ယခင်သုတေသန ၁၅၀ ကျော်ကို ခွဲခြမ်းစိတ်ဖြာထားသော ကျယ်ပြန့်သောနောက်ဆုံးပေါ် ပြန်လည်သုံးသပ်ချက် (၂၀၂၅)။ ဗီတာမင် C ၏ ဆေးဝါးဆိုင်ရာသွေးတွင်းအာနိသင် (20-30 mM) ကို သွေးကြောထဲသို့ တိုက်ရိုက်ပေးအပ်ခြင်းဖြင့်သာ ရရှိနိုင်ကြောင်း သတ်မှတ်ပြီး၊ ဤအာနိသင်တွင် လုပ်ဆောင်သော ကင်ဆာဆန့်ကျင်သည့် ယန္တရား လေးခုကို အောက်တွင် စုစည်းဖော်ပြထားသည်။ ခီမိုကုသမှု (သို့) ရောင်ဂျင်ကုထုံးနှင့် ပေါင်းစပ်ပါက ဘေးထွက်ဆိုးကျိုးများ (ပင်ပန်းနွမ်းနယ်ခြင်း၊ အစာအိမ်ဆိုင်ရာလက္ခဏာများကဲ့သို့) လျော့ကျနိုင်ခြေနှင့် ဘဝအရည်အသွေးတိုးတက်ကောင်းမွန်နိုင်ခြေကို ပြသနေသော ဆေးဘက်ဆိုင်ရာဒေတာများ စုဆောင်းနေကြောင်းလည်း မှတ်သားထားပြီး၊ အကောင်းဆုံးပမာဏ၊ ပေးအပ်နည်းနှင့် လူနာရွေးချယ်မှုနှင့်ပတ်သက်၍ ပိုမိုကြီးမားသော စမ်းသပ်မှုများ လိုအပ်ဆဲဖြစ်ကြောင်း နိဂုံးချုပ်ထားသည်။

① ရွေးချယ်ကာ ဆဲလ်အဆိပ်သင့်စေခြင်း (Pro-oxidant အကျိုးသက်ရောက်မှု)
သံဓာတ်ငွေ့ (iron ion) များနှင့် ဓာတ်ပြု၍ hydrogen peroxide (H₂O₂) ထုတ်လုပ်ပြီး၊ catalase လုပ်ဆောင်ချက်နည်းသော ကင်ဆာဆဲလ်များကို ရွေးချယ်ကာ ထိခိုက်စေသော်လည်း ပုံမှန်ဆဲလ်များအပေါ် အဆိပ်သင့်မှုကို နည်းပါးစေသည်။
② Epigenetic ထိန်းညှိမှု
TET enzyme များကို ပြန်လည်လှုပ်ရှားစေ၍ အကျိတ်ဟန့်တားဗီဇများ ထုတ်ဖော်မှုကို ပြန်လည်ရရှိစေသည်။ ပုံမမှန် DNA methylation ကို ပြောင်းပြန်ပြုလုပ်ခြင်းဖြင့် အကျိတ်ဆဲလ်များ ပွားများခြင်းကို နှိမ်နင်းသည်။
③ ကင်ဆာဆက်စပ် အချက်ပြလမ်းကြောင်းများ နှိမ်နင်းခြင်း
အကျိတ်၏ပြင်းထန်မှုကို ထိန်းချုပ်သည့် transcription factor ဖြစ်သော HIF-1α ကို ဖျက်စီးသည့် enzyme ၏ coenzyme အဖြစ် လုပ်ဆောင်ပြီး၊ အကျိတ်ကြီးထွားမှုနှင့် ကူးစက်ပျံ့နှံ့မှုအချက်ပြမှုကို ပိတ်ဆို့ပေးသည်။
④ ကိုယ်ခံအားလှုံ့ဆော်ခြင်း
NK ဆဲလ်နှင့် T ဆဲလ်အစရှိသော ကိုယ်ခံအားဆဲလ်များ၏ လုပ်ဆောင်ချက်ကို အားကောင်းစေပြီး၊ အကျိတ်ဝန်းကျင်ရှိ ကိုယ်ခံအားတုံ့ပြန်မှုကို မြှင့်တင်ပေးသည်။

Intravenously administered vitamin C as cancer therapy: three cases

ကျောက်ကပ်ကင်ဆာ၊ ဆီးအိမ်ကင်ဆာနှင့် B-ဆဲလ် lymphoma ဖြစ်ပွားသူ ဖြစ်ရပ်သုံးခုတွင် မြင့်မားသောဗီတာမင် C ဆေးရည်ငွေ့ကုထုံးပြီးနောက် မမျှော်လင့်ထားသော ရေရှည်အသက်ရှင်ခြင်း (သို့) အကျိတ်ကျုံ့ခြင်းကို တွေ့ရှိရကြောင်း သတင်းပို့ထားသည်။ စံနှုန်းကုသမှုနှင့်အတူ ပံ့ပိုးပေးသောအခန်းကဏ္ဍကို ဆွေးနွေးသော ဦးဆောင်ဖြစ်ရပ်မှတ်တမ်းတစ်ခုဖြစ်သည်။

Intravenous High-Dose Vitamin C in Cancer Therapy

မြင့်မားသောဗီတာမင် C ဆေးရည်ငွေ့၏ ဘေးကင်းလုံခြုံမှုနှင့် ခံနိုင်ရည်ကို Phase I/II စမ်းသပ်မှုများစွာတွင် အတည်ပြုထားပြီး၊ ခီမိုကုသမှုနှင့်ပေါင်းစပ်ပါက ဘဝအရည်အသွေးတိုးတက်ကောင်းမွန်နိုင်ခြေနှင့် ဘေးထွက်ဆိုးကျိုးလျှော့ချနိုင်ခြေကို ပြသသည်။ ပုံမှန်ဆဲလ်များအပေါ် အဆိပ်သင့်မှုမရှိဘဲ ကင်ဆာဆဲလ်များအပေါ် ရွေးချယ်ကာ လုပ်ဆောင်သည့် ယန္တရားကိုလည်း စုစည်းဖော်ပြထားသည်။

💉
ပမာဏ
ပမာဏနှင့် ဈေးနှုန်း

ပမာဏကို သင့်ရည်မှန်းချက်အလိုက် ရွေးချယ်ပါသည်။ အလှအပနှင့် ကိုယ်ခံအားမြှင့်တင်ရန်ရည်ရွယ်ချက်အတွက် 25g သည် ပုံမှန်ဖြစ်ပြီး၊ ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးရန်အတွက် 50-75g ကို တစ်ပတ်လျှင် တစ်ကြိမ် (သို့) နှစ်ကြိမ် ပေးအပ်ခြင်းသည် ပုံမှန်ဖြစ်သည်။ မြင့်မားသောပမာဏ (50g သို့မဟုတ်ထိုထက်ပို၍) မပေးအပ်မီ G6PD (glucose-6-phosphate dehydrogenase) ချို့တဲ့မှုအတွက် စစ်ဆေးမှု လိုအပ်ပါသည် (ပထမအကြိမ်တွင်သာ)။

ခံတွင်း (အကန့်အသတ်)
0.2mM ခန့် (အထက်ကန့်သတ်ချက်ရှိသည်)
ဆေးရည်ငွေ့ 25g
အလှအပနှင့် ကိုယ်ခံအားမြှင့်တင်ခြင်း
ဆေးရည်ငွေ့ 50g
ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးခြင်း — စံနှုန်း
ဆေးရည်ငွေ့ 75g
ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးခြင်း — မြင့်မားသောပမာဏ
🌿 ဆေးစစ်ဆေးမှုလုပ်ငန်းစဉ်

ပထမဆုံးအကြိမ်လာရောက်ချိန်တွင် မေးမြန်းစဉ် သင့်ရည်မှန်းချက်များ၊ လက်ရှိကုသမှုအခြေအနေနှင့် ဆေးဝါးများကို မေးမြန်းပါမည်။ 50g (သို့) ထိုထက်ပို၍ ခံယူလိုပါက G6PD စစ်ဆေးမှု (သွေးထုတ်ခြင်း) ကို ဦးစွာပြုလုပ်ပါမည်။

ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးရန် လာရောက်သူများအတွက် လက်ရှိကုသဆရာဝန်၏ ကုသမှုစီမံချက်နှင့် စစ်ဆေးမှုဒေတာများကို ယူဆောင်လာပါက ပိုမိုတိကျသော အကြံပြုချက်များ ပေးနိုင်ပါသည်။ အဓိကကုသဆရာဝန်နှင့် ဆွေးနွေးရန် ခက်ခဲသူများထံမှလည်း တိုင်ပင်မှုများကို ကြိုဆိုပါသည်။

အခြားဆေးရည်ငွေ့များနှင့်လည်း ပေါင်းစပ်နိုင်ပါသည်။

ကင်ဆာကာကွယ်ကုသမှုပံ့ပိုးခြင်း၊ အလှအပ (သို့) ကိုယ်ခံအားပံ့ပိုးခြင်းနှင့်ပတ်သက်၍ ဦးစွာ လွတ်လပ်စွာ ဆက်သွယ်နိုင်ပါသည်။
